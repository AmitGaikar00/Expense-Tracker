## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd UI
  cd Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  UI - npm start
  Backend - npm run dev
```
